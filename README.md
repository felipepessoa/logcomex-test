# Teste Felipe Pessoa para Logcomex

## Para executar o projeto deve seguir os seguintes passos

1.  `cd backend/docker`
2.  `docker build -t logcomex/backend .`
3.  Ir para raiz do projeto `../../`
4.  `docker compose up -d`

O projeto estará pronto para executar

- As rotas de api se encontram em `https://logcomex.localhost/api/v1`
- O front end se encontra em `https://logcomex.localhost`
